# Gullsmed Jensens Eftf.

Statisk onepager for Gullsmed Jensens Eftf.

## Lokal visning

Start en enkel lokal server fra prosjektmappen:

```sh
python3 -m http.server 4174
```

Åpne deretter:

```txt
http://localhost:4174/
```

## Publisering

Siden kan publiseres gratis med GitHub Pages fra `main`-branchen.
